OPTIONS

Based on registering
- users can register up until the vote is initiated
- only registered users can vote during voting phase
- voting ends when all registered users have voted

Based on max voters
- maximum number of voters is set (perhaps can be changed by chairperson)
- users can register and vote both before and during voting phase
- voting ends when max voters number of users have voted

Based on time
- users can register and vote both before and during voting phase
- timer starts when voting starts
- voting ends when timer ends
